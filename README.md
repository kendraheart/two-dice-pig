## General Information
-  A Java GUI implementation of the two-player dice game two dice pig. In this game players take turns rolling dice while accumulating pooints with each roll. A player has won if they reach 100 or more points. You can select to play against a human or computer player. During each turn, a player can decide to roll the dice, hold their score, or quit. If a player rolls a one, they lose their turn and if a player rolls snake eyes, they lose their turn and their accumulated points. If playing against a computer player, you have the ability to choose a standard or custom holding strategy.

## Features
- 3 game modes: player vs player, player vs simple computer, player vs smart computer
- Ability to restart, quit, or exit the game
- Dynamic game stats
- Ability to rematch once the game is over
